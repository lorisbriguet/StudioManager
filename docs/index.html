StudioManager

A lightweight desktop app for freelance designers to manage clients, projects, invoices, and finances.

Download for macOS View on GitHub

Everything you need, nothing you don't

Client Management

Full client database with contacts, per-client language and discount settings. Everything linked to their projects and invoices.

Project Tracking

Organize projects by client with hierarchical tasks and subtasks. Inline editing, due dates, drag-and-drop reordering.

Calendar

Month and week views with tasks on their due dates. One-directional iCloud Calendar sync on macOS.

Invoicing with QR-Bill

Generate professional PDF invoices with Swiss QR payment slips. Automatic numbering, overdue detection, and status tracking.

Expense Tracking

Categorized expenses with receipt upload. Drag-and-drop PDFs, supplier autocomplete with smart autofill.

Finance Overview

Real-time P&L with charts. Revenue, expenses, and net result following Swiss tax declaration format.

Built to be fast and lightweight

Native performance with Tauri v2. No cloud, no subscriptions, no bloat.

Tauri v2 React 19 TypeScript SQLite Tailwind CSS Rust

Made for Swiss freelancers

Purpose-built for independent workers in Switzerland, with features that match how you actually work and file taxes.
